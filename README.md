Sky Gap Escape is a 3D side-scrolling platform game built with Unreal Engine 5.6. Collect coins, avoid obstacles, jump across gaps, and reach the finish line. The game includes two playable levels, audio, and a complete user interface. Successfully tested on Ubuntu Linux.
